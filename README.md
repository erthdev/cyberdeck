# Cyberdeck Project

A personal project to build a functional, portable cyberdeck from scratch. Starting from an old HP Compaq 8200 SFF, this project evolved, pivoted, and reset into something more focused, which is a compact mini PC-powered cyberdeck housed inside a repurposed butane stove case.

This is a living document. Plans change. Hardware gets fried. Ideas get scrapped and replaced with better ones. That is the point.

---

## What this is

This project began as an attempt to turn a full desktop motherboard into a portable machine. That idea hit a wall fast — wrong hardware, wrong scale, wrong approach. After stepping back and rethinking everything, the cyberdeck found its correct form: a low-power Intel Celeron-based mini PC board, existing RAM and storage, and a cleaned-up butane stove case as the enclosure.

It is small. It is portable. It proves the concept.

---

## Hardware

| Component | Details |
|---|---|
| Board | Mini PC — Intel Celeron 2950M, 2.00GHz, 2C/2T |
| RAM | 8GB DDR3L SO-DIMM |
| Storage | M.2 SATA SSD |
| Display Output | VGA / HDMI via board |
| Connectivity | USB 2.0, Gigabit LAN |
| Power | External DC power adapter |
| Enclosure | Repurposed butane stove case |

---

## Build Philosophy

A cyberdeck does not have to be powerful. It has to be intentional.

This build is about proving that a portable and functional, hand-built computer can come from salvaged parts, creative enclosures, and the willingness to figure things out as you go. Every mistake is documented. Every move and pivot is explained. Nothing is hidden.

Built by an IT student who wanted to go beyond theory and actually make something.
